# 篮球串关 (Parlay) 核心风控与实战策略指南

篮球的让分盘和大小分盘是由精算师通过海量数据建立的，其精准度远超足球。因此，在篮球串关中，“长串（3串1及以上）”的容错率极低，职业玩家的串关策略必须建立在**“极致的风控”**和**“同场关联逻辑”**之上。

---

## 一、 篮球串关的“三大死穴”（绝对避坑）

1. **死穴一：双胆双大热（强强联合）**
   - **表现**：把今天全网最看好的两支强队（如：凯尔特人赢盘 + 掘金赢盘）串在一起。
   - **原因**：篮球的“大热必死”效应极强。当所有资金倒向一边时，庄家会利用深盘的**“后门掩护 (Backdoor Cover)”**（即强队垃圾时间放水，赢球输盘）来收割。
   - **对策**：稳健单**绝对禁止**串两场大热的让分盘。必须采用“一冷一热”或“一让分一大小分”进行交叉。

2. **死穴二：无视体能红线的深盘**
   - **表现**：重注一支处于**客场背靠背 (B2B)**或**长途客场之旅**的强队打穿深盘（如 -8.5）。
   - **原因**：篮球第四节的体能崩盘是断崖式的，防守轮转跟不上极易被弱队一波流追回盘口。在欧篮（EuroLeague）的**双赛周第二场**更是如此。

3. **死穴三：只看场均得分串大小分**
   - **表现**：两支场均120分的球队相遇，无脑串大分。
   - **原因**：大小分的核心是**回合数 (Pace)**和**防守效率**。如果两支快节奏球队相遇，庄家会把总分开到极其夸张的位置（如 245.5），一旦有一方三分失准，大分必死。

## 二、 职业级串关策略 (实战模型)

### 策略 1：同场关联投注 (Same Game Parlay 逻辑延伸)
如果你极度看好一场比赛的某个走势，可以通过逻辑关联来绑定串关（注：国内竞彩不支持同场串，此逻辑用于寻找多场的共性）。
- **逻辑 A：[快节奏强队赢盘] 绑定 [大分]**
  - 如果你认为步行者（快节奏、三分大队）能打穿盘口，说明他们今天手感极佳，此时这场比赛打出**大分**的概率会极具相关性。
- **逻辑 B：[铁血防守队赢盘] 绑定 [小分]**
  - 如果你认为尼克斯/森林狼这种靠防守绞杀的球队能赢盘，那必然是一场泥地肉搏战，打出**小分**的概率极高。

### 策略 2：2串1 矩阵防守打法（推荐）
这是最科学的资金管理方式。
- **配置**：挑选 1 场【极度稳健的大小分】 + 1 场【中度信心的让分盘】。
- **为什么避开双让分？** 让分盘极易因为最后1分钟的无意义犯规战术（点球战术）被绝杀，而大小分盘通常在比赛还剩3分钟时就已经大局已定。

### 策略 3：单日赛事极少时的“交叉盘避险”
- **场景**：今天只有 2-3 场 NBA/CBA 赛事。
- **动作**：全国赌狗无赛可买，资金会极其集中。此时庄家的杀伤力最大。**必须强制防冷**！在你的串关方案中，必须有一场去博取**受让方（下盘）赢盘**。

## 三、 针对大模型串关生成的优化建议 (Prompt 植入点)

如果你要优化 `bball_predictor.py` 中的 `generate_parlays` 提示词，建议强制大模型遵守以下排盘规则：

1. **【严禁长串】**：除娱乐方案外，核心主推方案必须是 **2串1**。
2. **【强制异构组合】**：在生成 2串1 方案时，优先使用 **[一场让分盘] + [一场大小分盘]** 的异构组合，降低同类盘口的系统性风险。
3. **【规避垃圾时间风险】**：当挑选让分极深（如让 10.5分以上）的比赛作为串关胆材时，必须在理由中注明已排除了“后门掩护”的风险。
4. **【体能一票否决制】**：严禁将处于“客场背靠背”的球队作为稳健串关的让分上盘胆材。
