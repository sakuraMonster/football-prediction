# 竞彩篮球预测功能开发计划

## 1. 核心需求分析
用户希望新增“竞彩篮球”的预测模块，其整体业务流程与现有的足球模块保持一致：
1. 从官方渠道抓取今日篮球赛事及赔率数据。
2. 将数据组装并提交给大语言模型（LLM）进行分析预测。
3. 展现单场比赛的深度分析结果。
4. 提供多场比赛组合的“智能串单推荐”功能。

## 2. 具体实施步骤

### 第一步：开发篮球数据爬虫 (`src/crawler/jclq_crawler.py`)
- **目标源**：500彩票网竞彩篮球频道 (`https://trade.500.com/jclq/`)。
- **解析字段**：
  - 基本信息：赛事编号、联赛名称、主队、客队、比赛时间。
  - 盘口与赔率：让分盘口(`data-rangfen`)、大小分盘口(`data-yszf`)。
  - 赔率数据：胜负(SF)、让分胜负(RFSF)、大小分(DXF)的实时赔率。

### 第二步：开发篮球大模型预测器 (`src/llm/bball_predictor.py`)
- **专属 Prompt 设计**：篮球的分析逻辑与足球不同，需为其定制专属的 System Prompt。核心风控与分析点将包括：
  - 明星球员伤停影响。
  - 赛程疲劳度（如背靠背比赛）。
  - 主客场胜率差异。
  - 机构对“让分”和“大小分”深盘的诱导意图。
- **功能方法**：实现 `predict()` 单场预测方法，以及 `generate_parlays()` 篮球专属的智能串单方法。

### 第三步：数据库扩展 (`src/db/database.py`)
- **新增数据表**：创建 `BasketballPrediction` 表，字段结构类似足球，但需适应篮球的特性（如记录让分、大小分结果）。
- **新增方法**：添加保存篮球预测、查询篮球预测的数据库操作方法。

### 第四步：构建篮球专属前端界面 (`src/pages/3_Basketball.py`)
- **全新页面**：在 Streamlit 侧边栏新增“🏀 竞彩篮球”菜单。
- **功能模块**：
  - **全局控制区**：一键抓取最新篮球赛事并调用大模型进行全局预测。
  - **汇总与串关区**：展示今日所有篮球赛事的精简预测列表，并提供“🏀 生成篮球智能串关方案”的按钮。
  - **单场详情区**：通过折叠面板展示每场比赛的详细数据（让分、大小分）及 LLM 深度分析报告。支持单场重新预测。

### 第五步：数据缓存与联调测试
- **本地缓存**：引入 `data/today_bball_matches.json` 作为前端界面的数据流转介质。
- **测试验证**：运行爬虫确保能正确解析500网的篮球 HTML 结构；测试大模型对篮球 Prompt 的响应质量。

## 3. 预期成果
完成后，系统侧边栏将出现独立的篮球预测模块，用户可在此界面一键获取当天的 NBA/CBA 等竞彩赛事的深度分析报告及高回报串关方案，且足球与篮球模块数据相互独立，互不干扰。
