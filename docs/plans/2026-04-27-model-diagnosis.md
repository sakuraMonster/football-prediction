## 二、核心缺陷深度分析

### 🔴 缺陷 1：Prompt 严重膨胀导致"注意力稀释"（最致命）

**问题机制**：
- LLM 存在"Lost in the Middle"效应：超长上下文下，模型对中间部分的注意力会急剧衰减
- 当 89 处出现"绝对/极其/致命/严禁/必须/极度"等极端副词时，这些词会相互贬值——模型无法区分哪些是"真正的致命规则"与"普通的强化语气"
- 类比：一个人同时收到 89 条"极其重要"的信息，他会全部忽略

**✅ 修复方案：三层规则体系**
- **P0 硬约束（5条）**：绝对的逻辑自洽规则（上下盘定义、主客不颠倒、让球逻辑自洽、分析结论一致、数量限制）
- **P1 盘型决策树**：先判定盘型（平手/浅盘/中盘/深盘/超深盘），再匹配对应规则，消除互斥
- **P2 场景辅助（6条）**：联赛校准、赛季末战意、双线作战、德比/死敌、换帅、防线伤停

---

### 🔴 缺陷 2：规则互斥与自相矛盾（"精神分裂"根源）

**问题**：同一场景下存在相互矛盾的指令：

| 场景 | 规则A (推上盘) | 规则B (推下盘) |
|:---|:---|:---|
| 深盘+低水 | "深盘穿透力觉醒" → 推让胜 | "一球盘屠杀预期陷阱" → 推让平/让负 |
| 退盘 | "顺势散热" → 推正路 | "降盘麻木症" → 防冷下盘 |
| 平半高水 | "平半铁律" → 推主胜 | "浅盘引流陷阱" → 防上盘 |

**根本原因**：规则都是通过复盘"贴补丁"加上去的，没有做全局一致性校验。

**✅ 修复方案：盘型决策树**
重构为决策树结构——先判定盘型（平手/浅盘/中盘/深盘/超深盘），再匹配对应规则。同一盘型下互斥的规则被合并为统一判断标准（如"区分阻上 vs 诱上"），非此即彼。

---

### 🔴 缺陷 3：盘口驱动的执行与理论脱节

**理论正确性**：盘口驱动原则本身是正确的。机构掌握的信息 > 公开基本面，盘口是机构意图的最终表达。

**原执行问题**：
1. LLM 缺乏实时市场数据（只有初盘+即时盘2个点）
2. 无法执行"三重交叉验证"
3. "盘力值计算"是伪命题 —— LLM 没有量化模型，只能凭"感觉"判断深或浅

**✅ 修复方案**：
1. **新增 `_detect_odds_change`**：自动解析澳门/Bet365初盘→即时盘变化，输出结构化摘要，如 `澳门: 退盘(半球→平手/半球)；上盘水位↑升水(0.95→1.00)`
2. **去掉LLM无法执行的伪指令**（如"计算盘力值"）
3. **新增 `_get_league_hint`**：为不同联赛注入特性提示，帮助LLM快速校准预期

---

### 🟡 缺陷 4：NSPF提取成功率仅22%

**原因**：
1. LLM 输出格式不统一（新旧Prompt混用）
2. 旧格式：分析报告开头有大段开场白，NSPF行埋藏在深层
3. 部分预测全文没有独立的不让球推荐行

**✅ 修复方案（多层次兜底）**：
1. 行级匹配增强：支持 `**不让球**：` / `* 不让球：` 等格式
2. `line_clean`预清洗：去除 `**` `*` 标记后再匹配
3. 全文fallback搜索：在报告尾部搜索推荐关键词
4. 终极兜底：提取带权重的胜/平/负模式（如 `胜(80%)`）

**效果**：**22% → 93%**（+322%提升），近100条历史数据验证通过。

---

### 🟡 缺陷 5：`is_correct` 字段从未被更新

**原因**：`run_post_mortem.py` 在复盘时计算了 `is_correct` 但只在 Python 变量中，从未写回数据库。

**✅ 修复**：在复盘循环中增加 `UPDATE match_predictions SET is_correct = ... WHERE id = ...`，并在复盘结束时 `db.session.commit()`。

---

### 🟡 缺陷 6：法甲/美职联/德乙准确率崩塌

**原因**：
1. **法甲**：防守联赛DNA + 进球数低 + 盘口诱盘手段与其他联赛不同
2. **美职联**：被标记为"大球联赛"但 MLS 主场优势极强且季后赛赛制特殊
3. **德乙**：次级联赛盘口操纵更严重，数据透明度低

**✅ 修复**：`_get_league_hint` 为每个联赛注入特异性提示：

| 联赛类型 | 提示内容 |
|:---|:---|
| 防守型（法甲/葡超/西甲/法乙） | 深盘优先防小球和赢球输盘，忌盲目推3+球 |
| 大球型（美职联/澳超/荷甲） | 攻击力碾压时敢于推大比分和让胜 |
| 次级联赛（英冠/德乙/日乙） | 盘口操纵更严重，退盘升水高度重视 |
| 北欧（挪超/瑞超） | 主场优势极显著，赛季初冷门频发 |

## 三、盘口驱动原则的正确性判断

### ✅ 理论正确
盘口数据确实比公开基本面包含更多信息（内幕、资金流向、机构对冲需求）。世界顶级博彩模型都以赔率变化为核心特征。

### ⚠️ 原实现差距（已修复）

| 理论上需要的 | 修复前 | 修复后 |
|:---|:---|:---|
| 多时间点赔率序列 | 初盘 + 即时盘（2个点） | 初盘 + 即时盘 + 结构化异动摘要 |
| 多家机构赔率分歧度 | 澳门 + Bet365（2家） | 2家 + 异动对比 |
| 盘力值量化模型 | LLM 凭语感估算 | 已删除此伪指令 |
| 联赛特性校准 | 无 | `_get_league_hint` 注入 |

### 📋 结论
**盘口驱动原则正确。核心优化方向：将盘赔分析从 LLM的"语感推理"转变为"结构化数据 + 盘型决策树匹配"的双通道体系。**
