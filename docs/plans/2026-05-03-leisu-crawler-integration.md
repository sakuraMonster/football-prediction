# 雷速体育爬虫与预测模型数据增强方案

> **Goal:** 通过 Playwright 浏览器自动化，从雷速体育（live.leisu.com）自动获取比赛伤停信息、进球分布、积分排名、历史交锋、半全场数据等，替代现有 500.com 不全的伤停数据源，并增强预测模型的输入维度。

**Analysis Date:** 2026-05-03
**Status:** 📋 待执行

## 一、背景与动机

### 1.1 当前系统的伤停数据缺陷

当前系统唯一伤停来源是 `odds_crawler.py` 中的 `fetch_match_details()`，从 `500.com` 的"预计阵容"HTML 表格抓取。该数据存在以下问题：

1. **覆盖率低**：大量比赛在 500.com 上没有预计阵容数据（输出"暂无详细伤停数据"）
2. **格式粗糙**：只有球员姓名，无位置、出场数、进球数等量化信息，LLM 无法判断严重程度
3. **无伤停量化**：当前只存储为纯文本 `"主队伤病: 张三, 李四 | 客队停赛: 王五"`

### 1.2 雷速体育的数据优势

雷速体育是覆盖最全的国内体育数据平台之一，每场比赛详情页包含：

| Tab/模块 | 数据内容 | 对预测模型的价值 |
|:---|:---|:---|
| **阵容伤停** | 预计首发 + 伤病/停赛名单（含位置、出场数、进球数） | **核心价值**：量化核心伤停影响 |
| **进球分布** | 上下半场进球时间分布（0-15', 16-30', ...） | D12 半场进球率的数据源 |
| **积分排名** | 联赛实时积分榜 + 主客场战绩 | D7 战意评估的数据源 |
| **历史交锋** | 近 N 次交锋的比分、胜负、半全场结果 | D6 交锋规律 + 半全场预测 |
| **近期战绩** | 近 6~10 场比分、盘路、进球数 | D4/D5 近期趋势的数据源 |
| **半全场** | 历史交锋的半全场结果分布 | 半全场预测模型的数据增强 |
| **赔率对比** | 多家机构的胜平负/让球/大小球 | 现有数据已有，可做交叉验证 |
