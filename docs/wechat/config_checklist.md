# 公众号后台配置清单（仅导向私聊）

1. 自定义菜单
- 今日前瞻 → 最新图文
- 今日扫盘列表 → 合集页
- 今日串关推荐 → 公域样例页
- 联系顾问 → 二维码单页（仅此项）
- 关于我们 → 品牌介绍/往期精选、业务合作

2. 自动回复（关键词与文案见 auto_replies.md）
- “开通/周卡/月卡/年卡/价格/续费/会员” → 私聊导向+二维码
- “串关/低倍/高倍/方案” → 今日串关样例+二维码
- “使用/说明/声明/风险/合规” → 使用说明与声明

3. 永久页面准备
- 联系顾问（见 contact_page.md）
- 使用说明与合规声明（见 disclaimer.md）
- 今日串关推荐（公域样例，见 daily_parlay_template.md）

4. 图文模板与 CTA
- 在日更文章中插入 CTA 片段（见 cta_snippets.md）
- 固定在文末插入免责声明

5. 素材准备
- 私人微信二维码（清晰、更新；命名规范，便于复用）
- 统一封面模板（颜色与字体一致）

6. 发布与复盘
- 固定发布时间：比赛日前 16:00–19:00；临场前再发简报
- 每周一次复盘总结（公域），透明记录命中与失误
- 关注指标：关键词触发率、菜单点击率、关注→私聊转化、私聊→支付转化、次日留存
