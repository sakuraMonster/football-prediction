# 老泊 · 私聊沟通SOP（简洁版）

欢迎
- 老泊在这儿。稳一点还是高倍？说下偏好，我按你的口味发今天的思路。

需求澄清
- 偏好：低倍稳一点 / 高倍敢冲一点
- 常看联赛/球队：____
- 可接受回撤：____；单次注码：____

交付节奏
- 早盘一版 + 临场一版；重要异动补简报
- 内容包含：方向 + 风控提示；不承诺收益

风控与提示
- 固定注码、控制仓位；拒绝重注或梭哈
- 连续失误会及时复盘与节奏调整

收尾示例
- 好的，我先把今天稳一点的低倍二串一思路发你，理由和注意点一起附上；高倍想要的话也给你一版，量力而行就行。

轻量声明
- 内容是数据思路分享，理性参考就好。
