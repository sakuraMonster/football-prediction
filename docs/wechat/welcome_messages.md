# 老泊 · 首次欢迎语模板

定位：亲和、接地气、不端着；专业但不讲术语堆砌。用于两处：1）关注后自动回复；2）私人微信通过好友后的第一句。

关注后自动回复
- 版本A（标准）  
嗨，我是老泊。平时就爱琢磨数据和球，发点靠谱的前瞻、扫盘和串关思路。想看今天的重点，回我“今日”；要样例串关，回“串关”；想看复盘，回“复盘”。别急，慢慢看，稳一点总没错。

- 版本B（亲和）  
来啦兄弟，我是老泊。爱看数据，也爱少说废话。这里不吹牛，只讲方向和风控。想要今天的看法回“今日”，看看串关思路回“串关”，复盘我也放这儿，回“复盘”就行。

- 版本C（简短）  
老泊在这儿。今日看法回“今日”，串关样例回“串关”，复盘回“复盘”。理性一点，走得更远。

私人微信通过好友后的第一句
- 版本A（标准）  
嗨，我是老泊。你更偏稳一点还是想搏高倍？说下偏好，我按你的口味发今日的方向和注意点。

- 版本B（简短）  
老泊到了。稳胆还是高倍，选一个，我把今天的思路发你。

轻量提示（可选）  
老泊碎念：看球图开心，做单讲节奏。别重注，别上头。
